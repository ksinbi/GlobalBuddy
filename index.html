Project

Global Buddy

[ 문제 정의 ]

언어 학습을 할 때 시간과 장소 제한으로 인한 어려움을 해결하기 위해 간단하게 사이트로 대하하며 학습을 할 수 있게 하는 것이 목표입니다.

[ 아키텍처/설계 ]

  • 데이터베이스 (Firestore): 사용자가 원하는 상대를 생성할 때 만들어진 페르소나를 관리하기 위해 사용합니다.
  • 인증 (Firebase Authentication): 사용자 인증을 하기 위해 Firebase Authentication을 사용합니다.
  • AI 연동 (Gemini API): 대화에서 틀린 내용을 실시간 피드백을 위해 사용합니다.

[ 핵심 기능 ]

사용자 맞춤 페르소나

사용자 맞춤 프롬포트로 원하는 성격의 상대를 만들 수 있습니다. (LLM)

지능형 피드백

AI를 이용하여 사용자가 문법에 맞지 않은 문장을 구사하면 알맞은 문법으로 답할 수 있도록 피드백을 줍니다.

[ 나의 역할 및 기여 ]

  • 'Global Buddy' 아이디어 기획
  • LLM 기반 페르소나 구축 및 프롬프트 엔지니어링
  • GitHub를 이용한 버전 관리

[ 결과 및 성과 ]

페르소나 일관성 유지 - LLM 프롬프트 최적화를 통해 사용자가 설정한 페르소나가 대화 내내 일관되게 유지되도록 구현했습니다.

[ 회고 (Retrospective) ]

배운 점

LLM을 사용하여 어떻게 사용자의 물음에 최적화된 응답을 할 수 있는지, 그리고 일관성 있는 대화 경험을 제공하기 위한 프롬프트 설계의 중요성을 알게 되었습니다.
